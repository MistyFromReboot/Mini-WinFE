  ЛИЦЕНЗИОННОЕ СОГЛАШЕНИЕ

  Настоящее лицензионное соглашение является юридическим соглашением 
  между Вами (физическим или юридическим лицом), и автором данного 
  программного обеспечения (ПО) Дмитрием Сидоровым. Используя ПО, Вы 
  принимаете условия Соглашения.

  1. ПРЕДОСТАВЛЯЕМЫЕ ПРАВА

  1.0. РЕДАКЦИИ ПО

  Объем Ваших прав на использование ПО зависит от РЕДАКЦИИ ПО (типа 
  лицензии) и количества лицензионных копий, предоставленных Вам. 

  РЕДАКЦИЯ, количество копий и количество поддерживаемых платформ ОС 
  указаны в Вашем заказе лицензии. РЕДАКЦИЯ, количество копий, срок 
  лицензии также отображаются в окне ПО "О программе" (в меню ПО 
  "Справка"). Если количество копий не указано явно, то предоставлена 
  одна (1) лицензия. Если срок лицензии не указан, то он не ограничен.

  1.1. РЕДАКЦИЯ "FREE EDITION"

  К редакции "FREE EDITION" применяются только пункты 1.1, 5 и 6.

  Можно использовать любое число копий "FREE EDITION" в некоммерческих 
  целях, включая образовательные цели учреждений и ознакомление. 
  Допускается восстановление файлов только из текущей панели и 
  с ограничением до 4000 файлов за раз (одной командой).

  Копии "FREE EDITION" можно создавать и распространять (в том числе в 
  составе платных комплексов) при выполнении всех условий: ПО получено 
  непосредственно у автора или на сайтах ПО; не передаются ключи и иные 
  компоненты других РЕДАКЦИЙ ПО; состав и компоненты ПО не изменены 
  (кроме обратимого перекодирования); происхождение, авторство, условия 
  данного соглашения доступны конечному пользователю в явном виде.

  1.2.1. РЕДАКЦИИ "EXPRESS" И "PERSONAL": ТОЛЬКО ЛИЧНОЕ ИСПОЛЬЗОВАНИЕ

  Редакции "EXPRESS" и "PERSONAL" дают право использовать ПО только в 
  личных некоммерческих целях. Запрещается любое использование ПО, 
  прямо или косвенно приводящее к получению выгоды.

  1.2.2. РЕДАКЦИЯ "STANDARD": ИСПОЛЬЗОВАНИЕ ТОЛЬКО ДЛЯ СОБСТВЕННЫХ НУЖД

  Редакция "STANDARD" дает право использовать ПО частным лицам и 
  организациям только для собственных нужд. Запрещается оказание услуг 
  по восстановлению данных для третьих лиц. В том числе, ПО не могут 
  использовать любые организации, а также их сотрудники и иные лица, 
  предоставляющие услуги по восстановлению данных.

  1.3. ЗАПУСК И ХРАНЕНИЕ КОПИЙ ПО

  Вы можете одновременно запускать экземпляры ПО в пределах количества 
  предоставленных копий лицензии (число незавершенных процессов не 
  должно превышать количество копий лицензии).

  В общей сложности Вы можете создать и хранить на две (2) копии ПО 
  больше, чем количество копий лицензии. Одна копия может включать 
  несколько версий ПО в одном хранилище данных.

  2. АКТИВАЦИЯ

  Активация связывает запуск ПО с конкретной аппаратной конфигурацией 
  ЭВМ и разблокирует полную функциональность ПО. Активация требует 
  отправки кода идентификации оборудования на сайты активации ПО для 
  получения кода разблокировки.

  Активация может быть отклонена в случае запуска ПО более чем на 
  трех (3) аппаратных конфигурациях в год на одну копию лицензии. 
  Смена процессора или материнской платы (включая изменение серийного 
  номера в ПЗУ), использование дополнительной виртуализации означают 
  смену аппаратной конфигурации. При некоторых других аппаратных или 
  программных изменениях может потребоваться (и допускается) повторная 
  активация ПО.

  В случае истечения срока лицензии Вы можете продолжать использование 
  ПО в прежней среде, однако, никакая поддержка, в том числе повторная 
  активация, не предоставляется.

  3. ОБНОВЛЕНИЯ ПО

  Если поддержка имеющейся версии прекращается, Вы можете использовать 
  следующую доступную версию, при этом новое соглашение заменяет 
  предыдущее только в пунктах, уточняющих условия и явно расширяющих 
  Ваши права. В иных случаях новые версии могут использоваться только 
  полностью на условиях новых соглашений.

  Вы также можете использовать предыдущие доступные версии ПО на 
  условиях данного соглашения. Однако, запуск предыдущих версий ПО 
  (с учетом номера до первой точки) приравнивается к запуску на другом 
  оборудовании с применением ограничений пункта 2.

  4. АПГРЕЙД ПО

  Апгрейд - изменение комплектации, числа копий лицензии или РЕДАКЦИИ. 
  Для апгрейда ПО необходимо иметь исходную лицензию, для которой 
  предоставляется апгрейд. При апгрейде новое лицензионное соглашение 
  заменяет собой предыдущее, и Вы больше не можете использовать ключи 
  и иные компоненты, предоставленные по условиям предыдущей лицензии.
